# Custom Copilot Agents

This repository includes an example custom agent definition to help guide Copilot for tasks associated with the React task manager app.

## Agent: Task Manager Assistant
- **Name:** `task-manager`
- **Purpose:** Focus on features for the task manager app (add/delete/toggle tasks, UI/UX improvements, accessibility, React best practices).

### How to use
1. Open the Copilot panel in VS Code.
2. Select the **Task Manager Assistant** agent (if available).
3. Ask questions like:
   - "Add a priority field to each task"
   - "Make the task list persist in localStorage"
   - "Improve keyboard accessibility for task actions"

---

## Agent: Code Reviewer
- **Name:** `code-reviewer`
- **Purpose:** Review code for correctness, readability, and best practices; provide concise feedback and suggested improvements.

### How to use
1. Open the Copilot panel in VS Code.
2. Select the **Code Reviewer** agent.
3. Ask questions like:
   - "Review this component for performance issues"
   - "Is there a simpler way to handle task filtering?"
   - "Are there any accessibility issues with this UI?"

---

> Note: Copilot agents are an evolving feature. If you don’t see the agent listed, the default Copilot assistant will still work for this repo.
